TextQuest

Description

"TextQuest" is an interactive, text-based adventure game set in the world of Hunter x Hunter, specifically during the Greed Island arc. The player will navigate the game using typed commands, exploring locations, interacting with NPCs, and utilizing game mechanics like the Book of Cards and Nen abilities. The game begins with the player entering Greed Island through the use of a game console and ring.

Notes

  • Genre: Fantasy Adventure Card Game.
  • Setting: The world of Greed Island (Hunter x Hunter).
  • Mechanics: Text-based exploration, inventory management, and puzzle-solving.

Main Features

  1. Interactive Commands:
    • Movement (e.g., go north, enter cave).
    • Inventory management (e.g., book, gain).
    • Use of Book of Cards to materialize and gain abilities.
  2. World Navigation:
    • Exploration of various locations like the Hub, cities, Forests, and Caves.
    • Interaction with NPCs who offer clues, items, or quests.
  3. Story Progression:
    • Choices and actions will influence the outcome of puzzles and quests.

Game Outline

0. Menu

  • Options: Start Game, Load Game, Options, Exit.
  • Start Game: Prompts the player to enter a name and begin their adventure.
  • Load Game: Load previously saved progress.
  • Options: Text size and color.
  • Exit: Quit the game.

1. Start

  • Player is prompted to enter their name:
    "Please enter your name: "
  • Once the name is entered:
    "Player <player_name> puts the ring on and uses Nen on the game console to enter Greed Island."

2. Location 1: Game Introduction Hub

  • Description:
    "You spawn in front of a woman sitting in a chair, her head adorned with a peculiar helmet. Her name is Elena."
  • NPC Interaction:
    • Elena introduces the player to the rules of the game and acts as a tutorial guide.
    • Dialogue:
      "Welcome to Greed Island. I am Elena, your guide... "
    • Elena will explain what game is about, this will include the slots in the book, how card are used .. etc.
  • Tutorial:
    • Elena explains the basic commands and core mechanics:
      • Open the Book of Cards: Type book to view available cards.
      • Use a card: While holding a card, type gain to materialize its ability.
    • Reference Hunter x Hunter Chapter 127 for more on how the Book of Cards works.

Additional Commands

  • help for possible commands.
  • book to open the book == check current items and cards.
  • gain to materialize the card in hand == use a specific item from the inventory e.g. gain 2 to use card two.
  • look to inspect your surroundings.
  • talk to [character] to interact with characters.
  • move [direction] to navigate the world.
  • examine [object] to get detailed information about a specific object.
  • take [item] to pick up an item from the current location.
  • drop [item] to remove an item from hand of from book.
  • save to save the current game state.
  • load to load a previously saved game state.
  • quit to exit the game.
  • Gyo use Gyo (Gyo consumes Nen points.)
  • status to display the player's current stats (Health, Nen points, etc.)
  • look around for a more detailed description of the environment.
